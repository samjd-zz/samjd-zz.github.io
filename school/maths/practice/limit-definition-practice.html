üî¨ Derivative by Limit Definition Practice

Limit Definition of Derivative:
$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$ When: Proving derivative rules or finding rate of change from first principles

üìù General Process:

Step 1: Find f(x+h) by substituting (x+h) into the function
Step 2: Calculate f(x+h) - f(x)
Step 3: Divide by h
Step 4: Simplify the expression
Step 5: Take the limit as h ‚Üí 0

üåü Basic Level - Linear and Quadratic Functions

1 Find the derivative of $$f(x) = 3x$$ using the limit definition
Solution:

Step 1: Find f(x+h)
$$f(x+h) = 3(x+h) = 3x + 3h$$ Step 2: Calculate f(x+h) - f(x)
$$f(x+h) - f(x) = (3x + 3h) - 3x = 3h$$ Step 3: Divide by h
$$\frac{f(x+h) - f(x)}{h} = \frac{3h}{h} = 3$$ Step 4: Take the limit
$$f'(x) = \lim_{h \to 0} 3 = 3$$ Answer: f'(x) = 3
2 Find the derivative of $$f(x) = x^2$$ using the limit definition
Solution:

Step 1: Find f(x+h)
$$f(x+h) = (x+h)^2 = x^2 + 2xh + h^2$$ Step 2: Calculate f(x+h) - f(x)
$$f(x+h) - f(x) = (x^2 + 2xh + h^2) - x^2 = 2xh + h^2$$ Step 3: Divide by h
$$\frac{f(x+h) - f(x)}{h} = \frac{2xh + h^2}{h} = \frac{h(2x + h)}{h} = 2x + h$$ Step 4: Take the limit
$$f'(x) = \lim_{h \to 0} (2x + h) = 2x + 0 = 2x$$ Answer: f'(x) = 2x
3 Find the derivative of $$f(x) = 5$$ using the limit definition
Solution:

Step 1: Find f(x+h)
$$f(x+h) = 5$$ (constant function) Step 2: Calculate f(x+h) - f(x)
$$f(x+h) - f(x) = 5 - 5 = 0$$ Step 3: Divide by h
$$\frac{f(x+h) - f(x)}{h} = \frac{0}{h} = 0$$ Step 4: Take the limit
$$f'(x) = \lim_{h \to 0} 0 = 0$$ Answer: f'(x) = 0
The derivative of any constant is 0!
4 Find the derivative of $$f(x) = 2x + 7$$ using the limit definition
Solution:

Step 1: Find f(x+h)
$$f(x+h) = 2(x+h) + 7 = 2x + 2h + 7$$ Step 2: Calculate f(x+h) - f(x)
$$f(x+h) - f(x) = (2x + 2h + 7) - (2x + 7) = 2h$$ Step 3: Divide by h
$$\frac{f(x+h) - f(x)}{h} = \frac{2h}{h} = 2$$ Step 4: Take the limit
$$f'(x) = \lim_{h \to 0} 2 = 2$$ Answer: f'(x) = 2

‚ö° Intermediate Level - Higher Powers and Fractions

5 Find the derivative of $$f(x) = x^3$$ using the limit definition
Solution:

Step 1: Find f(x+h)
$$f(x+h) = (x+h)^3 = x^3 + 3x^2h + 3xh^2 + h^3$$ Step 2: Calculate f(x+h) - f(x)
$$f(x+h) - f(x) = (x^3 + 3x^2h + 3xh^2 + h^3) - x^3$$ $$= 3x^2h + 3xh^2 + h^3$$ Step 3: Divide by h
$$\frac{f(x+h) - f(x)}{h} = \frac{3x^2h + 3xh^2 + h^3}{h}$$ $$= \frac{h(3x^2 + 3xh + h^2)}{h} = 3x^2 + 3xh + h^2$$ Step 4: Take the limit
$$f'(x) = \lim_{h \to 0} (3x^2 + 3xh + h^2) = 3x^2 + 0 + 0 = 3x^2$$ Answer: f'(x) = 3x¬≤
6 Find the derivative of $$f(x) = \frac{1}{x}$$ using the limit definition
Solution:

Step 1: Find f(x+h)
$$f(x+h) = \frac{1}{x+h}$$ Step 2: Calculate f(x+h) - f(x)
$$f(x+h) - f(x) = \frac{1}{x+h} - \frac{1}{x}$$ Find common denominator:
$$= \frac{x - (x+h)}{x(x+h)} = \frac{x - x - h}{x(x+h)} = \frac{-h}{x(x+h)}$$ Step 3: Divide by h
$$\frac{f(x+h) - f(x)}{h} = \frac{-h}{x(x+h)} \cdot \frac{1}{h} = \frac{-1}{x(x+h)}$$ Step 4: Take the limit
$$f'(x) = \lim_{h \to 0} \frac{-1}{x(x+h)} = \frac{-1}{x \cdot x} = -\frac{1}{x^2}$$ Answer: f'(x) = -1/x¬≤
7 Find the derivative of $$f(x) = x^2 - 3x$$ using the limit definition
Solution:

Step 1: Find f(x+h)
$$f(x+h) = (x+h)^2 - 3(x+h)$$ $$= x^2 + 2xh + h^2 - 3x - 3h$$ Step 2: Calculate f(x+h) - f(x)
$$f(x+h) - f(x) = (x^2 + 2xh + h^2 - 3x - 3h) - (x^2 - 3x)$$ $$= 2xh + h^2 - 3h$$ Step 3: Divide by h
$$\frac{f(x+h) - f(x)}{h} = \frac{2xh + h^2 - 3h}{h}$$ $$= \frac{h(2x + h - 3)}{h} = 2x + h - 3$$ Step 4: Take the limit
$$f'(x) = \lim_{h \to 0} (2x + h - 3) = 2x - 3$$ Answer: f'(x) = 2x - 3
8 Find the derivative of $$f(x) = \sqrt{x}$$ using the limit definition
Solution:

Step 1: Find f(x+h)
$$f(x+h) = \sqrt{x+h}$$ Step 2: Calculate f(x+h) - f(x)
$$f(x+h) - f(x) = \sqrt{x+h} - \sqrt{x}$$ Rationalize: Multiply by conjugate
$$= \frac{(\sqrt{x+h} - \sqrt{x})(\sqrt{x+h} + \sqrt{x})}{\sqrt{x+h} + \sqrt{x}}$$ $$= \frac{(x+h) - x}{\sqrt{x+h} + \sqrt{x}} = \frac{h}{\sqrt{x+h} + \sqrt{x}}$$ Step 3: Divide by h
$$\frac{f(x+h) - f(x)}{h} = \frac{h}{\sqrt{x+h} + \sqrt{x}} \cdot \frac{1}{h} = \frac{1}{\sqrt{x+h} + \sqrt{x}}$$ Step 4: Take the limit
$$f'(x) = \lim_{h \to 0} \frac{1}{\sqrt{x+h} + \sqrt{x}} = \frac{1}{\sqrt{x} + \sqrt{x}} = \frac{1}{2\sqrt{x}}$$ Answer: f'(x) = 1/(2‚àöx)

üöÄ Advanced Level - Complex Functions

9 Find the derivative of $$f(x) = \frac{1}{x^2}$$ using the limit definition
Solution:

Step 1: Find f(x+h)
$$f(x+h) = \frac{1}{(x+h)^2}$$ Step 2: Calculate f(x+h) - f(x)
$$f(x+h) - f(x) = \frac{1}{(x+h)^2} - \frac{1}{x^2}$$ Find common denominator:
$$= \frac{x^2 - (x+h)^2}{x^2(x+h)^2} = \frac{x^2 - (x^2 + 2xh + h^2)}{x^2(x+h)^2}$$ $$= \frac{-2xh - h^2}{x^2(x+h)^2}$$ Step 3: Divide by h
$$\frac{f(x+h) - f(x)}{h} = \frac{-2xh - h^2}{x^2(x+h)^2} \cdot \frac{1}{h}$$ $$= \frac{h(-2x - h)}{hx^2(x+h)^2} = \frac{-2x - h}{x^2(x+h)^2}$$ Step 4: Take the limit
$$f'(x) = \lim_{h \to 0} \frac{-2x - h}{x^2(x+h)^2} = \frac{-2x}{x^2 \cdot x^2} = \frac{-2x}{x^4} = -\frac{2}{x^3}$$ Answer: f'(x) = -2/x¬≥
10 Find the derivative of $$f(x) = 2x^2 + 5x - 1$$ using the limit definition
Solution:

Step 1: Find f(x+h)
$$f(x+h) = 2(x+h)^2 + 5(x+h) - 1$$ $$= 2(x^2 + 2xh + h^2) + 5x + 5h - 1$$ $$= 2x^2 + 4xh + 2h^2 + 5x + 5h - 1$$ Step 2: Calculate f(x+h) - f(x)
$$f(x+h) - f(x) = (2x^2 + 4xh + 2h^2 + 5x + 5h - 1) - (2x^2 + 5x - 1)$$ $$= 4xh + 2h^2 + 5h$$ Step 3: Divide by h
$$\frac{f(x+h) - f(x)}{h} = \frac{4xh + 2h^2 + 5h}{h}$$ $$= \frac{h(4x + 2h + 5)}{h} = 4x + 2h + 5$$ Step 4: Take the limit
$$f'(x) = \lim_{h \to 0} (4x + 2h + 5) = 4x + 5$$ Answer: f'(x) = 4x + 5
11 Find the derivative of $$f(x) = x^4$$ using the limit definition
Solution:

Step 1: Find f(x+h) using binomial expansion
$$f(x+h) = (x+h)^4 = x^4 + 4x^3h + 6x^2h^2 + 4xh^3 + h^4$$ Step 2: Calculate f(x+h) - f(x)
$$f(x+h) - f(x) = (x^4 + 4x^3h + 6x^2h^2 + 4xh^3 + h^4) - x^4$$ $$= 4x^3h + 6x^2h^2 + 4xh^3 + h^4$$ Step 3: Divide by h
$$\frac{f(x+h) - f(x)}{h} = \frac{4x^3h + 6x^2h^2 + 4xh^3 + h^4}{h}$$ $$= \frac{h(4x^3 + 6x^2h + 4xh^2 + h^3)}{h}$$ $$= 4x^3 + 6x^2h + 4xh^2 + h^3$$ Step 4: Take the limit
$$f'(x) = \lim_{h \to 0} (4x^3 + 6x^2h + 4xh^2 + h^3) = 4x^3$$ Answer: f'(x) = 4x¬≥
12 Find the derivative of $$f(x) = \frac{2x}{x+1}$$ using the limit definition
Solution:

Step 1: Find f(x+h)
$$f(x+h) = \frac{2(x+h)}{(x+h)+1} = \frac{2x+2h}{x+h+1}$$ Step 2: Calculate f(x+h) - f(x)
$$f(x+h) - f(x) = \frac{2x+2h}{x+h+1} - \frac{2x}{x+1}$$ Find common denominator:
$$= \frac{(2x+2h)(x+1) - 2x(x+h+1)}{(x+h+1)(x+1)}$$ $$= \frac{2x^2 + 2x + 2xh + 2h - 2x^2 - 2xh - 2x}{(x+h+1)(x+1)}$$ $$= \frac{2h}{(x+h+1)(x+1)}$$ Step 3: Divide by h
$$\frac{f(x+h) - f(x)}{h} = \frac{2h}{(x+h+1)(x+1)} \cdot \frac{1}{h} = \frac{2}{(x+h+1)(x+1)}$$ Step 4: Take the limit
$$f'(x) = \lim_{h \to 0} \frac{2}{(x+h+1)(x+1)} = \frac{2}{(x+1)(x+1)} = \frac{2}{(x+1)^2}$$ Answer: f'(x) = 2/(x+1)¬≤
13 Prove that the derivative of $$f(x) = x^n$$ is $$nx^{n-1}$$ using the limit definition (general case)
Solution:

We need to prove: If f(x) = x^n, then f'(x) = nx^(n-1)

Step 1: Set up the limit definition
$$f'(x) = \lim_{h \to 0} \frac{(x+h)^n - x^n}{h}$$ Step 2: Use the binomial theorem
$$(x+h)^n = x^n + nx^{n-1}h + \binom{n}{2}x^{n-2}h^2 + ... + h^n$$ Step 3: Substitute into the limit
$$f'(x) = \lim_{h \to 0} \frac{x^n + nx^{n-1}h + \binom{n}{2}x^{n-2}h^2 + ... + h^n - x^n}{h}$$ Step 4: Simplify
$$= \lim_{h \to 0} \frac{nx^{n-1}h + \binom{n}{2}x^{n-2}h^2 + ... + h^n}{h}$$ $$= \lim_{h \to 0} \frac{h(nx^{n-1} + \binom{n}{2}x^{n-2}h + ... + h^{n-1})}{h}$$ $$= \lim_{h \to 0} (nx^{n-1} + \binom{n}{2}x^{n-2}h + ... + h^{n-1})$$ Step 5: Take the limit
As h ‚Üí 0, all terms with h vanish:
$$f'(x) = nx^{n-1} + 0 + 0 + ... + 0 = nx^{n-1}$$ Therefore: The power rule is proven! $$\frac{d}{dx}[x^n] = nx^{n-1}$$

üí° Tips for Success:

  • Always follow the 5-step process: f(x+h), subtract f(x), divide by h, simplify, take limit
  • Expand carefully: Use FOIL for (x+h)¬≤, and the binomial theorem for higher powers
  • Factor out h: Before taking the limit, always factor h from the numerator to cancel with the h in the denominator
  • For fractions: Find a common denominator before subtracting
  • For square roots: Multiply by the conjugate to rationalize
  • Check your algebra: Most errors come from arithmetic mistakes, not conceptual misunderstanding
  • The limit removes h: After simplifying, substitute h = 0 to get the final answer
  • Practice recognition: The result should match what you'd get using derivative rules!
